# Changelog

All notable changes to the **NES Light Theme** will be documented in this file.

## [1.0.3] - 2026-06-19

### Added
- Added the official theme logo/icon to the extension marketplace.

### Changed
- Customized global action buttons (e.g., Git Sync/Publish) to match the signature NES red branding (`#b11e14`).
- Updated the editor group split drop background to use a semi-transparent red instead of the default VS Code blue.
- Changed the sidebar and panel resize sash hover border color to the theme's standard red.

### Fixed
- Cleaned up unused comments and boilerplate code from the main theme JSON file for better performance and maintainability.

---

## [1.0.0] - 2026-06-13
- Initial release of the NES Light Theme.
- Warm light theme inspired by classic retro consoles.
